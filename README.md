## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Map Engine**: Mapbox GL JS + Deck.gl
- **UI Framework**: Material-UI (MUI)
- **State Management**: Zustand
- **Data Fetching**: TanStack Query
- **Backend**: Supabase
